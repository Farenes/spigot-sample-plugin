# Spigot Sample Plugin

Учебный плагин для сервера **Minecraft** на базе [Spigot API](https://www.spigotmc.org/) (Java). Проект собирается через **Maven** и демонстрирует типичную структуру плагина: главный класс, команды, слушатели событий и `plugin.yml`.

**Целевая версия API:** `1.21.11-R0.2-SNAPSHOT`.

## Требования

- JDK **22**
- Apache **Maven** 3.x
- Сервер Spigot/Paper, совместимый с указанной версией API

## Сборка

```bash
mvn clean package
```

Готовый JAR появится в каталоге `target/` (имя артефакта соответствует `artifactId` и версии из `pom.xml`).

Скопируйте JAR в папку `plugins` вашего сервера и перезапустите сервер или выполните перезагрузку плагинов (если поддерживается).

## Возможности

### Команды

| Команда | Описание |
|--------|----------|
| `/pos` | Без аргументов — выводит текущие координаты и углы обзора. С тремя числами `/pos x y z` — телепортирует игрока в указанную точку в текущем мире. |
| `/debug` | Переключает режим отладки для игрока: при движении координаты логируются в консоль сервера. |

### События (примеры)

- **Игрок:** лог при входе; при получении урона — сообщение, небольшой отскок вверх, эффекты и частицы; при включённом `/debug` — лог движения.
- **Блоки:** отмена физики песка/гравия под железным блоком; разрешение установки кактуса там, где иначе могло бы быть запрещено.

Код служит шаблоном для собственных плагинов; поведение можно менять в пакете `ru.vlsu.plugin`.

## Структура проекта

```
src/main/java/ru/vlsu/plugin/
├── SamplePlugin.java          # точка входа плагина
├── command/                   # исполнители команд
└── listener/                  # обработчики событий Bukkit
src/main/resources/
└── plugin.yml                 # метаданные и регистрация команд
```
